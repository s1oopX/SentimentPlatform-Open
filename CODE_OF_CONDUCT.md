# Code of Conduct

SentimentPlatform aims to keep collaboration respectful, professional, and constructive.

## Expected Behavior

- Be kind and specific when giving feedback.
- Focus on reproducible issues, clear proposals, and maintainable fixes.
- Respect different backgrounds and levels of experience.

## Unacceptable Behavior

- Harassment, discrimination, threats, or personal attacks.
- Publishing private data, credentials, local databases, model assets, or datasets.
- Disrupting project maintenance, CI, issues, or pull requests.

Maintainers may edit, hide, or remove content that violates this policy.
