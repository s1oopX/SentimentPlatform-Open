# Security Policy

Please do not disclose vulnerabilities, credentials, account data, or exploit details in public issues.

## Report Privately

Use GitHub Security Advisories where possible. Include affected modules, reproduction steps, impact, and suggested remediation.

## Sensitive Data

This repository must not contain:

- `.env` files or real secrets.
- Local demo accounts, phone numbers, verification codes, or database exports.
- Uploaded files, generated reports, backups, or logs.
- Model weights, training artifacts, Arrow files, or datasets.

If any such content is found, report it immediately and avoid spreading the affected commits.
